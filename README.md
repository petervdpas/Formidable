## 📖 About This Project

**Formidable** was a modular Electron desktop application for creating, managing, and rendering dynamic forms and Markdown documents from YAML-based templates. It combined a visual form editor with a Handlebars-style rendering engine, a virtual file system (VFS), profile switching, and optional Git/GiGot integration — designed for professionals who need structured content management, versioning, and auditability.

Over its lifetime it grew to include 20+ field types, an event-driven architecture, a hot-reloadable plugin system, and 50,000+ words of documentation. It served as the foundation and proving ground for what is now being continued in **[Formidable2](https://github.com/petervdpas/Formidable2)**.

### 🔧 New Stack in Formidable2

Formidable2 is a ground-up rebuild on a leaner, faster stack:

- **[Go](https://go.dev/)** — backend, file I/O, templating, and core logic
- **[Wails v3](https://wails.io/)** — native desktop shell (replacing Electron)
- **[Vue](https://vuejs.org/)** + **[TypeScript](https://www.typescriptlang.org/)** — reactive, typed frontend

The result: smaller binaries, lower memory footprint, native performance, and a cleaner separation between backend and UI than the Electron/vanilla-JS approach in this repo.
